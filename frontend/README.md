# Frontend (Vue 3 + Vite)

Фронт сайта «Калейдоскоп» — страницы: главная, каталог, новости, заявка.

**Установка и запуск (используй pnpm — npm на Windows может падать):**

```bash
pnpm install
pnpm run dev
```

Сайт откроется на http://localhost:5173/

Бэкенд (Django): админка и API на http://127.0.0.1:8000/admin/
